## 產品背景理解

### Smart Insight Engine 簡介
**產品定位**: 營養補充品數據分析 API 引擎
**核心功能**: 提供靈活的查詢介面，支援多維度數據分析

**架構特點**:
- **M/D/F/O 查詢結構** - Measure（指標）/ Dimensions（維度）/ Filters（篩選）/ Options（選項）
- **CISC/RISC 架構** - 用戶層複雜指令 vs 引擎層原子操作
- **13 個維度** - 1個簡單欄位 (Brand) + 12個分類法維度
- **10+ 種指標** - 計數、聚合、統計、分布分析
- **篩選邏輯** - ANY (OR) 和 ALL (AND) 組合邏輯
- **效能保護機制** - 維度限制、結果集大小控制、超時保護

**數據規模**:
- ~130K 產品數據集
- 13 個查詢維度
- 10+ 種分析指標
- 複雜的分類法層級結構
