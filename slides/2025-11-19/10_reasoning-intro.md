## 什么是 Reasoning？

**Reasoning = 推理 (从已知得出结论的能力)**

- **定义**：根据已有信息，推导出新结论的过程
- **类比**：像福尔摩斯破案 - 从线索推理出真相
- **核心能力**：逻辑思考、因果分析、模式识别

---

## Reasoning 的三种类型

### 1. 演绎推理 (Deductive)
从一般到特殊
```
所有人都会死 + 苏格拉底是人 → 苏格拉底会死
```

### 2. 归纳推理 (Inductive)
从特殊到一般
```
太阳今天升起 + 昨天也升起 → 太阳每天都会升起
```

---

## Reasoning 的三种类型 (续)

### 3. 溯因推理 (Abductive)
推测最可能的原因
```
观察：地上有水
可能：下雨 / 洒水 / 水管破了
结论：最可能是下雨（天空是阴的）
```

**在 AI 中的应用**：
- 代码调试：从错误信息推理问题原因
- 需求分析：从用户描述推理真正需求
- 系统设计：推理出最优架构方案
